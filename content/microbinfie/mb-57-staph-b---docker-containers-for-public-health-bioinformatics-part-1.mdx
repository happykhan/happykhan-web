# StaPH-B Containers: Discussion with Curtis Kapsak and Kevin Libuit

**Overview:**

In a recent discussion, the team engaged with Curtis Kapsak and Kevin Libuit to delve into the details of *StaPH-B* containers. These containers are recognized as valuable resources for many researchers and developers.

**Resources:**

- **StaPH-B GitHub Repository:**

  You can access the *StaPH-B* Docker builds code repository at the following link: [StaPH-B GitHub](https://github.com/StaPH-B/docker-builds).
  
- **StaPH-B DockerHub:**

  Explore the *StaPH-B* DockerHub container repositories here: [StaPH-B DockerHub](https://hub.docker.com/u/staphb).

- **Contribution Guide:**

  For those interested in contributing to the project, a comprehensive guide is available: [Contribute to StaPH-B](https://staph-b.github.io/docker-builds/contribute/).

These resources are essential tools for anyone looking to work with or contribute to *StaPH-B* container development.
