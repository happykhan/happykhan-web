# SARS-CoV-2 Genomic Developments

## Recent Discussions

In the past two weeks, we've explored the latest developments in SARS-CoV-2 genomics with expert Mads Albertsen, focusing particularly on updates from Denmark. For more detailed statistics on COVID-19 in Denmark, please visit: [Denmark COVID Stats](http://www.covid19genomics.dk/statistics).

The following are some of the notable tools and resources discussed:

- [Outbreak.info](https://outbreak.info/)
